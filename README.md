# factorio

https://site.tokiukaze.com/blog/factorio-headless-server-build/

## Download

https://www.factorio.com/download  

```sh
wget https://www.factorio.com/get-download/2.0.73/headless/linux64
tar xvf linux64
```

## Tunneling

```sh
playit
```

## Init

```sh
bin/x64/factorio --create test
```

## Run

```sh
bin/x64/factorio --start-server test
```
